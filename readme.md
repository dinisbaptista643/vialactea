# 🌌 Via Láctea — Estrelas Mais Próximas do Sol

Projeto desenvolvido em JavaScript com p5.js que representa uma visualização interativa da vizinhança solar dentro da Via Láctea. O programa mostra algumas das estrelas mais próximas do Sol utilizando dados astronómicos reais, animações suaves e uma interface gráfica interativa.

## 📚 Objetivo do Trabalho

O objetivo deste projeto é criar uma simulação visual educativa da nossa região da galáxia, permitindo explorar estrelas próximas do Sistema Solar através de visualização espacial em 2D, animações e interação com o utilizador.

## 🛠️ Tecnologias Utilizadas
JavaScript
p5.js
Canvas 2D
🧠 Conceitos Aplicados

O projeto utiliza vários conceitos importantes de programação gráfica e matemática, incluindo:

Conversão de coordenadas polares para cartesianas
Interpolação linear com lerp()
Oscilações e animações com sin()
Transformações gráficas com push() e pop()
Utilização de objetos e arrays para armazenar dados científicos
🌠 Funcionalidades
Visualização interativa da Via Láctea
Representação das estrelas mais próximas do Sol
Efeitos de brilho e pulsação
Fundo galáctico animado
Painel informativo com dados científicos
Zoom e movimentação da câmara
Seleção de estrelas com o rato
Escala visual em anos-luz
🔭 Dados Científicos

Os dados utilizados foram inspirados em catálogos astronómicos reais, incluindo informações da NASA e do Hipparcos Catalogue. O projeto apresenta estrelas como:

Sol
Próxima Centauri
Alpha Centauri A e B
Sirius A e B
Wolf 359
Estrela de Barnard
Ross 154
Ross 248
